# 嫦娥五号登月任务模拟 / Chang'e 5 Lunar Mission Simulation

交互式 3D 登月任务模拟，完整重现嫦娥五号从文昌发射到月壤取样返回的全部 13 个飞行阶段。

Interactive 3D lunar mission simulation that recreates all 13 flight phases of the Chang'e 5 mission — from Wenchang launch to lunar sample return.

## 任务概览 / Mission Overview

| 项目 | 数据 |
|------|------|
| 任务名称 | 嫦娥五号 (Chang'e 5) |
| 运载火箭 | 长征五号遥五 (CZ-5) |
| 发射场 | 海南文昌航天发射场 (19.6°N, 110.9°E) |
| 发射时间 | 2020-11-24 04:30 UTC |
| 着陆点 | 风暴洋吕姆克山 (43.06°N, 51.92°W) |
| 采样质量 | 1,731 克月壤 |
| 返回时间 | 2020-12-17 01:59 UTC |
| 总任务天数 | 23 天 |
| 总 Delta-V | ~17,800 m/s |

这是中国首次月球采样返回任务，也是自 1976 年苏联月球 24 号以来人类首次从月球带回样品。

## 使用方式 / How to Use

### 启动任务

点击底部时间栏右侧的 🚀 火箭图标按钮启动模拟。

Click the rocket icon button on the right side of the bottom timebar to start the simulation.

### 控制面板

任务启动后，底部显示控制面板：

| 控件 | 功能 |
|------|------|
| ▶️/⏸ 按钮 | 播放/暂停 |
| ◀️/▶️ 按钮 | 跳转到上一/下一阶段 |
| 速度选择器 | Auto（自动变速）或手动选择 1000x-50000x |
| 🎯 追踪按钮 | 重新锁定相机到飞行器 |
| 进度条 | 彩色分段，点击可跳转到任意阶段 |
| ✕ 按钮 | 关闭模拟 |

### 交互行为

- **鼠标滚轮**: 缩放视角（保持跟踪飞行器）
- **拖拽**: 旋转环绕视角（保持跟踪飞行器）
- **卫星/星球点击**: 任务期间禁用，防止误切视角
- **卫星/标签**: 任务期间自动隐藏，保持画面整洁

## 任务阶段 / Mission Phases

模拟包含完整的 13 个飞行阶段，每个阶段有独立的轨迹颜色、描述和速度增量数据。

| # | 阶段 | ID | 时长 | Delta-V | 颜色 | 描述 |
|---|------|----|------|---------|------|------|
| 1 | 发射升空 | `launch` | 500s (~8分) | 9,400 m/s | 🟠 橙红 | 从文昌发射，经历助推器分离、整流罩抛罩、级间分离 |
| 2 | 停泊轨道 | `parking` | 1,800s (~30分) | — | 🔵 青色 | 200km 近地圆轨道，系统检查 |
| 3 | 地月转移变轨 | `tli` | 360s (~6分) | 3,150 m/s | 🟡 黄色 | 上面级发动机点火，加速至 10.93 km/s |
| 4 | 地月转移飞行 | `transfer` | 403,200s (~4.7天) | 30 m/s | 🟢 绿色 | 滑行段，两次中途修正 |
| 5 | 近月制动 | `loi` | 1,020s (~17分) | 850 m/s | 🟣 紫色 | 减速入轨，对数螺旋接近 |
| 6 | 环月轨道 | `lunar_orbit` | 259,200s (~3天) | 45 m/s | 🔵 靛蓝 | 200km → 15km 椭圆轨道，轨返分离 |
| 7 | 动力下降 | `descent` | 720s (~12分) | 1,700 m/s | 🔴 红色 | 变推力发动机减速软着陆 |
| 8 | 月面作业 | `surface` | 68,400s (~19时) | — | 🟠 橙色 | 吕姆克山，钻取+表取月壤 |
| 9 | 月面起飞 | `ascent` | 360s (~6分) | 1,680 m/s | 🟧 浅橙 | 上升器起飞，中国首次月面发射 |
| 10 | 月球轨道交会对接 | `rendezvous` | 21,600s (~6时) | 50 m/s | 🟪 品红 | 自主对接，样品转移 |
| 11 | 月地转移变轨 | `tei` | 180s (~3分) | 830 m/s | 🔵 浅蓝 | 逃离月球引力 |
| 12 | 月地转移飞行 | `return_transfer` | 388,800s (~4.5天) | 15 m/s | 🟩 青绿 | 返回滑行段，轨返分离 |
| 13 | 半弹道跳跃式再入 | `reentry` | 600s (~10分) | 0 m/s | 🔴 红色 | 11.2 km/s 再入，跳跃减速，着陆内蒙古 |

**总模拟时长**: ~1,554,240 秒 (约 18 天)

## 发射子阶段 / Launch Sub-Phases

发射阶段（500 秒）内包含 7 个详细子阶段，每个子阶段在面板中实时显示注释说明：

| 时间 (T+) | 事件 | 说明 | 视觉效果 |
|-----------|------|------|----------|
| T+0s | 点火起飞 | 一级发动机及 4 个助推器同时点火 | — |
| T+12s | 程序转弯 | 火箭开始俯仰程序转弯 | — |
| T+174s | 助推器分离 | 4 个 3.35m 助推器脱落 | 🎬 助推器模型隐藏 |

## 自动速率模式 / Auto-Speed Mode

默认启用的 **Auto** 速率模式会根据当前阶段自动调整播放速率，实现约 **5 分钟**完整回放体验：

| 阶段 | 速率倍数 | 播放时长 | 叙事节奏 |
|------|---------|---------|---------|
| 发射升空 | 50x | ~10s | 慢速 — 清楚展示每个分离事件 |
| 停泊轨道 | 200x | ~9s | 中速 — 可见绕地弧线 |
| 地月转移变轨 | 50x | ~7s | 慢速 — 看清变轨点火 |
| 地月转移飞行 | 15,000x | ~27s | 加速 — 4.7 天压缩至半分钟 |
| 近月制动 | 100x | ~10s | 慢速 — 观察螺旋接近 |
| 环月轨道 | 10,000x | ~26s | 中速 — 可见多圈轨道 |
| 动力下降 | 50x | ~14s | 慢速 — 着陆细节 |
| 月面作业 | 5,000x | ~14s | 中速 — 月面停留 |
| 月面起飞 | 50x | ~7s | 慢速 — 月面发射 |
| 月球轨道交会对接 | 1,000x | ~22s | 中速 — 对接过程 |
| 月地转移变轨 | 30x | ~6s | 慢速 — 逃逸变轨 |
| 月地转移飞行 | 15,000x | ~26s | 加速 — 返程飞行 |
| 再入返回 | 50x | ~12s | 慢速 — 跳跃式再入 |

速率切换时使用平滑插值（lerp, dt × 3），避免速度突变。

用户可随时从 Auto 切换为手动速率（1000x - 50000x），也可切回 Auto。

### 太阳系暂停

任务启动时，全局太阳系模拟速度自动设为 1x（实时），避免行星和卫星高速运动干扰观看。任务结束后自动恢复之前的速率。

## 相机系统 / Camera System

### 自适应相机距离

任务启动时相机自动锁定飞行器。相机距离根据当前阶段平滑过渡：

| 阶段类型 | 相机距离 | 视觉效果 |
|----------|---------|---------|
| 发射/着陆 | 0.15 | 极近距 — 飞行器占屏幕 ~1/20，可见地表/月表 |
| 停泊/环月轨道 | 0.25-0.4 | 近距 — 可见轨道弧线和天体表面 |
| 变轨点火 | 0.2-0.5 | 中距 — 可见发动机效果和轨道变化 |
| 地月转移 | 16.0 | 远距 — 可见地月路径全貌 |

距离过渡使用指数平滑（`tD += (target - tD) * min(1, dt * 2)`），约 1 秒完成过渡。

### 交互

- **滚轮缩放**: 始终可用，调整相机距离
- **拖拽旋转**: 始终可用，改变观察角度
- **点击星球/卫星**: 任务期间禁用，防止误切焦点
- **追踪按钮**: 点击重新锁定飞行器

## 飞行器模型 / Spacecraft Models

### 长征五号运载火箭 (发射/停泊/TLI 阶段)

程序化生成的 3D 模型，使用 `MeshStandardMaterial` 配合 `emissive` 属性实现太空中的自发光效果。

| 组件 | 几何体 | 材质 | 名称标识 |
|------|--------|------|----------|
| 芯级 | 圆柱 (r=0.024-0.028, h=0.28) | 米白色 | `core` |
| 红色标志带 | 圆柱 (r=0.029, h=0.025) | 中国红自发光 | `stripe` |
| 级间分离环 | 圆柱 (r=0.030, h=0.008) | 深灰 | `sep_ring` |
| 整流罩 | 圆锥 (r=0.032, h=0.12) | 浅灰 | `fairing` |
| 整流罩过渡环 | 圆柱 | 深灰 | `fairing_ring` |
| 4×助推器 | 圆柱 (r=0.014-0.017, h=0.22) | 灰白 | `booster_0..3` |
| 4×助推器鼻锥 | 圆锥 | 灰白 | `booster_nose_0..3` |
| 4×助推器喷管 | 圆柱 | 深灰 | `booster_nozzle_0..3` |
| 芯级喷管 | 圆柱 (r=0.016-0.024) | 深灰 | `core_nozzle` |
| 上面级喷管 | 圆柱 (默认隐藏) | 深灰 | `upper_nozzle` |

命名部件用于发射子阶段的可视化分离（隐藏对应 mesh 模拟分离效果）。

### 嫦娥五号探测器 (转移及后续阶段)

| 组件 | 几何体 | 材质 |
|------|--------|------|
| 主体 | 八边形柱 (r=0.045) | 棕褐色 |
| 金色热控毯 | 八边形柱 (r=0.047) | 金色半透明，暖光自发光 |
| 2×太阳翼 | 矩形板 (0.18×0.06) | 深蓝色，6×2 细分 |
| 太阳翼支臂 | 圆柱 | 灰色 |
| 天线碟 | 圆柱 (r=0.035) | 银灰色 |
| 馈源喇叭 | 圆锥 | 灰色 |
| 4×着陆腿 | 圆柱+斜支撑+脚垫 | 灰色 |

### 发动机尾焰

双锥体结构：外层橙色锥体 + 内层亮黄色核心。每帧随机透明度实现闪烁效果。仅在有推力的阶段显示（launch, tli, loi, descent, ascent, tei）。

### 飞行器朝向

飞行器朝向使用有限差分法计算速度方向：对当前时刻 t 和 t+Δt 的位置取差值得到速度向量，飞行器模型的 +Y 轴（模型"前方"）指向该方向。

| 阶段 | 朝向行为 |
|------|---------|
| 发射 | 径向向外（远离地心），逐渐偏转向轨道面 |
| 轨道飞行 | 沿轨道切线方向 |
| 转移飞行 | 指向目的地方向 |
| 下降/上升 | 径向指向/远离天体表面 |

## 轨道力学 / Orbital Mechanics

### 缝合圆锥近似法 (Patched-Conic Approximation)

轨迹计算使用缝合圆锥近似法，根据飞行器所在位置选择参考天体：

- **地球附近**: 以地球为中心的开普勒轨道
- **转移段**: 地球-月球连线上的插值路径
- **月球附近**: 以月球为中心的开普勒轨道

### 缓动函数

| 函数 | 用途 |
|------|------|
| `easeInOutCubic` | 轨道半径过渡、升空路径混合 |
| `easeInOutQuint` | 地月转移弧线 — 更长的加减速尾部，模拟引力效应 |

### 近月制动对数螺旋

近月制动阶段使用对数螺旋 `r = a · e^(b·θ)` 代替线性半径缩减，产生更自然的入轨路径。

### 场景坐标系

| 参数 | 值 | 说明 |
|------|------|------|
| 1 场景单位 | 1 地球半径 | 6,371 km |
| 月球距离 | 60.3 场景单位 | ~384,400 km |
| 月球半径 | 0.27 场景单位 | ~1,737 km |
| 地球视觉比例 | `baseScale(Earth)` ≈ 0.046 | 真实比例下的显示大小 |

## 轨迹可视化 / Trajectory Visualization

### 轨迹线

- **缓冲区**: 5,000 个顶点的循环缓冲区
- **发射频率**: 每 0.05% 任务进度发射一个点（`totalDur × 0.0005`）
- **颜色**: 每个点继承当前阶段颜色，形成彩虹色轨迹
- **材质**: `LineBasicMaterial`，顶点着色，80% 不透明度
- **截锥体剔除**: 禁用（`frustumCulled = false`），确保远距离可见

## 技术架构 / Technical Architecture

### 文件结构

```
src/
├── data/
│   └── lunarMission.ts          # 任务数据：阶段定义、子阶段、速率表、相机距离表
├── simulation/
│   ├── lunarMission.ts          # 轨迹计算引擎：缝合圆锥法、缓动函数、朝向计算
│   └── lunarMissionVisuals.ts   # 3D 模型：火箭、探测器、尾焰、轨迹线
└── App.tsx                      # 主循环：动画帧更新、UI 面板、相机控制、状态管理
    index.css                    # 样式：面板、按钮、进度条
```

### 数据流

```
用户操作 → Window API (__startLunarMission / __jumpLunarPhase / ...)
    ↓
状态更新 → lunarMissionRef (useRef, 非 React 渲染循环)
    ↓
每帧计算 → computeMissionState(elapsed, earthPos, moonPos, earthScale, earthRotY)
    ↓
场景更新 → 飞行器位置/朝向/缩放 + 尾焰 + 轨迹点 + 相机跟踪
    ↓
UI 更新 → React 状态 (节流至每 10 帧): 阶段索引、进度、经过时间
```

### 性能优化

- **模型预创建**: 火箭和探测器模型在场景初始化时创建，阶段切换时交换可见性
- **轨迹循环缓冲**: 5,000 点上限，超出后覆盖最早的点
- **React 更新节流**: 每 10 帧更新一次 React 状态，避免 60fps 全速渲染引起不必要的 re-render
- **卫星隐藏**: 任务期间自动隐藏所有卫星、标签、选择框，减少渲染负担
- **朝向计算**: 使用 `_noHeading` 标志避免递归（有限差分需要两次位置计算）

### 简化与近似

本模拟为教育演示目的，包含以下简化：

- **轨道力学**: 使用缝合圆锥近似而非完整的 N 体问题求解
- **转移轨道**: 近霍曼转移近似，非精确的限制性三体问题解
- **时间压缩**: 各阶段时长为真实值，但空间位置为可视化简化
- **发射方位**: 从文昌纬度发射，方位角简化为平滑混合至轨道面
- **月球位置**: 使用模拟中的实时月球位置，非 2020 年历史位置
